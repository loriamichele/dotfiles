Shell setup script
==================

## Requirements

- Vim 8.1+
- Tmux 2.8+

## Installation

1. Clone the repository in your HOME/.dotfiles directory (this is important!):

    $ git clone --recursive https://github.com/loriamichele/dotfiles.git ~/.dotfiles

2. Go to the cloned repository

    $ cd ~/.dotfiles

3. Execute the setup script:

    $ ./setup.sh
